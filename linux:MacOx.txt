some unix/ MacOS X commands

ls  ----view all directory

ls -l  ---view directory in list.

ls -lah  - - - View the size of the file.

clear  ---clears everything

cd name --go inside a directory

mkdir name --make a directory

pwd  ----shows the path of the directory

rmdir name --removes directory

cd ..  -- goes back by one step 

cd --- takes you back to the home directory
